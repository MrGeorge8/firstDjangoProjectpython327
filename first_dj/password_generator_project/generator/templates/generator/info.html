Info

На главной странице пользователю предоставляется форма с опциями для настройки генерации пароля.
Пользователь может выбрать длину пароля из предложенного диапазона и установить опции включения заглавных букв, цифр и специальных символов.
После выбора параметров, пользователь нажимает кнопку Generate Password.

Request processing

Когда пользователь нажимает "Generate Password", выполняется функция генерация пароля. Функция создает список символов, начиная с букв в нижнем регистре.
Если выбрана опция uppercase, добавляются заглавные буквы.
Если выбрана опция numbers, добавляются цифры.
Если выбрана опция special, добавляются специальные символы.
Затем функция создает пароль, выбирая случайные символы в соответствии с выбранными параметрами.
